# 🏎️ Auto Engine Car Service - Integrated Workshop System

[![Laravel](https://img.shields.io/badge/Laravel-FF2D20?style=for-the-badge&logo=laravel&logoColor=white)](https://laravel.com)
[![TailwindCSS](https://img.shields.io/badge/Tailwind_CSS-38B2AC?style=for-the-badge&logo=tailwind-css&logoColor=white)](https://tailwindcss.com)
[![Vibe](https://img.shields.io/badge/Theme-Tokyo_Drift-orange?style=for-the-badge)](https://github.com/muhamadrizalprasetyo)

Sistem manajemen bengkel terintegrasi yang menghubungkan portal pelanggan publik dengan dashboard admin bengkel. Dibuat khusus untuk modernisasi operasional bengkel **Auto Engine Car Service** (Est. 2010) dengan estetika *Underground Tokyo Drift*.

---

## 🚀 Fitur Utama

### 📱 User Side (Mobile-First)
- **Live Inventory:** Cek stok sparepart langsung dari database.
- **Online Booking:** Reservasi servis lewat HP tanpa ribet.
- **Consultation:** Link cepat konsultasi via WhatsApp.
- **Feedback System:** Form saran dan kritik setelah booking untuk evaluasi pelayanan.

### 💻 Admin Side (Sistem Eksekutif)
- **Dashboard Statistik:** Pantau pendapatan dan stok kritis dalam satu layar.
- **Manajemen Reservasi:** Kelola antrean masuk (Terima/Selesaikan).
- **POS Kasir:** Input transaksi pelanggan bengkel secara langsung.
- **Laporan Keuangan:** Rekap data transaksi bulanan.

---

## 🛠️ Tech Stack
- **Backend:** Laravel 10
- **Frontend:** Tailwind CSS (Tokyo Drift Dark Theme)
- **Database:** MySQL (Laragon)
- **Icons:** Material Icons & Lucide React

## 🔧 Cara Menjalankan Proyek
1. Clone repo ini.
2. Jalankan `composer install`.
3. Setup `.env` dan jalankan `php artisan key:generate`.
4. Jalankan `php artisan migrate` untuk menyiapkan database.
5. Akses via Laragon di `http://auto-engine-web.test/landing`.
